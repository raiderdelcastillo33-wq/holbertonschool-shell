# Shell I/O Redirections and Filters

This project contains Bash scripts that practice input/output redirections, pipelines, and text filters.

## Scripts

- `0-hello_world` prints "Hello, World".
- `1-confused_smiley` displays a confused smiley.
- `2-hellofile` displays the content of `/etc/passwd`.
- `3-twofiles` displays the content of `/etc/passwd` and `/etc/hosts`.
- `4-lastlines` displays the last 10 lines of `/etc/passwd`.
- `5-firstlines` displays the first 10 lines of `/etc/passwd`.
- `6-third_line` displays the third line of the file `iacta`.
- `7-file` creates a file with a special-character name containing text.
- `8-cwd_state` writes the result of `ls -la` into a file.
- `9-duplicate_last_line` duplicates the last line of the file `iacta`.
- `10-no_more_js` deletes regular `.js` files in the current directory and subdirectories.
- `11-directories` counts directories and subdirectories.
- `12-newest_files` displays the 10 newest files in the current directory.
- `13-unique` prints words that appear exactly once from a sorted input.
- `14-findthatword` displays lines containing the word `root` from `/etc/passwd`.
- `15-countthatword` counts lines containing the word `bin` from `/etc/passwd`.
- `16-whatsnext` displays lines containing `root` and three lines after them from `/etc/passwd`.
- `17-hidethisword` displays lines from `/etc/passwd` that do not contain `bin`.
- `18-letteronly` displays lines from `/etc/ssh/sshd_config` that start with a letter.
- `19-AZ` replaces selected characters using `tr`.
- `20-hiago` removes selected characters using `tr`.
- `21-reverse` reverses input text.
- `22-users_and_homes` displays users and their home directories from `/etc/passwd`.
- `23-empty_casks` finds empty files and directories and prints only their names.
- `24-gifs` finds `.gif` files, removes their extension, and sorts the names.
- `25-acrostic` builds an acrostic from the first character of each input line.
- `26-the_biggest_fan` parses TSV web server logs and displays the 11 most active hosts or IP addresses.
